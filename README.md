# wts-node

Node.js client for [WTS system](https://github.com/chemdemo/wts-monit).

### Installation

``` bash
npm install wts-node && npm i
```

### Configuration

Assign group for this client.

``` bash
cd wts-node && vim conf.js
```

### debug

``` bash
node index.js
```

### deploy(via pm2)

``` bash
pm2 start pm2_deploy.json
```
